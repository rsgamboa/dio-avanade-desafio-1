## Funções Basicas 

<br>

### *Anvançar* - O avatar seguirá em frente.

<hr>

> moveForward(); 

<br>

### *Virar para Esquerda* - O avatar irá virar para esquerda. 

<hr>

> turnLeft();

<br>

### *Virar para Direita* - O avatar irá virar para direita. 

<hr>

> moveRight(); 

<br>

### *Destruir* - O avatar irá destruir o bloco. 

<hr>

> destroyBlock();

<br>

### *Tosar* - O avatar irá tosar o animal para coletar material.

<hr>

> shear(); 

<br>

### *Repetir* - Este script faz que você possa repetir a execução da função mais de uma vez. *"Ex: Avançar 4 casas".*

<hr>

> for (var count = 0; count < 4; count++) {
>    moveForward();
>} 

<br>

### *Plantar* - O avatar irá plantar.

<hr>

> plantCrop();

<br>

### *Colocar Bloco* - O avatar irá inserir blocos.

<hr>

> mplaceBlock("planksBirch");

<br>

### *"Se"* *"Faça"* - Se existir lava, faça colocar pedras adiante.

<hr>

> ifLavaAhead(function() {
>  placeBlockAhead("cobblestone");
>});

## Instale o Git no Windows

<hr>

> - Navegue até o instalador Git para Windows mais recente e baixe a versão mais recente.
> - Depois que o instalador for iniciado, siga as instruções fornecidas na tela do assistente de configuração do Git até que a instalação seja concluída.
> - Abra o prompt de comando do Windows (ou Git Bash se você optou por não usar o prompt de comando padrão do Git do Windows durante a instalação do Git).
> - Digite git versionpara verificar se o Git foi instalado.


> #### Nota: 
>
> *git-scmé um recurso popular e recomendado para baixar o Git para Windows. A vantagem de baixar o Git de git-scmé que o download começa automaticamente com a versão mais recente do Git incluída no prompt de comando recomendado Git Bash,. A fonte de download é o mesmo instalador do Git para Windows, conforme mencionado nas etapas acima.*

## Instale o Git no Linux

<hr>

>   Curiosidade: o Git foi originalmente desenvolvido para a versão do sistema operacional Linux! Portanto, só faz sentido que seja fácil de configurar para rodar no Linux. Você pode instalar Gitno Linux por meio da ferramenta de gerenciamento de pacotes que vem com sua distribuição.
>
>
> #### Debian / Ubuntu: 
>
>+ Os pacotes Git estão disponíveis usando apt.
>
>+ É uma boa ideia verificar se você está executando a versão mais recente. Para fazer isso, navegue até a >linha de comandos shell e execute o seguinte comando para garantir que tudo está up-to-date: sudo apt-get >update.
>
>+ Para instalar o Git, execute o seguinte comando: sudo apt-get install git-all.
>
>+ Uma vez que a saída do comando foi concluída, você pode verificar a instalação digitando: git version.
>
>
> #### Fedora: 
>
> Os pacotes Git estão disponíveis usando dnf.
>+ Para instalar o Git, navegue até a linha de comandos shell e execute o seguinte comando: sudo dnf install git-all.
>+ Uma vez que a saída do comando foi concluída, você pode verificar a instalação digitando: git version.

> #### Nota: 
> *você pode baixar as versões adequadas do Git e ler mais sobre como instalar em sistemas Linux específicos, como instalar o Git no Ubuntu ou Fedora, na documentação do git-scm.*
